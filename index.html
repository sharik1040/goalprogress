Goal Progress Tracker:

  • Type a goal(f.e. do morging exercise everyday) in TARGET field.
  • Type a number of days, you want to explore your progress in COUNTER field.
  • BE CAREFUL!In this version, your progress calendar starts directly from today, which means that you start doing your target right from today.
  • If you type 0, then you will delete your progress and start date.
  • If you type number less than you prev counter, then you calendar will be decreased.
  • If you type number higher than you prev counter, then you calendar will be increased.
  • After you will enter a number of days, you will get calendar(tracker).
  • Click on calendar day in order to mark it done.A done item will become green.
